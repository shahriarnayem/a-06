# DigiTools marketplace.

A website for buying digital tools that works well on all devices and was made with React.js. Look through and buy digital tools to speed up your work.


# Technologies used
React.
DaisyUI and Tailwind CSS
Toastify React
Icons for React
JavaScript (ES6+)
JSON

# Main Features
Interactive Cart System: Add, take away, and check out items. The total price changes in real time, and the number of items in the cart shows in the navbar.
Tab-Based Navigation: Use smooth animations to switch between the Products grid and the Cart list.
Toast Notifications: Instead of alerts, all actions (add, remove, checkout) show styled notifications.
